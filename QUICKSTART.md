# Быстрый старт

## Запуск игры

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите сервер:
```bash
python main.py
```

3. Браузер откроется автоматически по адресу http://127.0.0.1:5000

## Подключение с других устройств

После запуска сервер покажет локальный IP адрес, например:
```
Локальный адрес: http://192.168.1.100:5000
```

Другие устройства в той же локальной сети (Wi-Fi/Ethernet) могут подключиться по этому адресу:
- Смартфоны
- Планшеты
- Другие компьютеры

Важно: убедитесь, что брандмауэр Windows разрешает входящие подключения на порт 5000

## Мобильные устройства

Игра полностью адаптирована для мобильных устройств:
- Смартфоны (портретная и альбомная ориентация)
- Планшеты
- Touch-управление для рисования
- Адаптивный интерфейс с оптимизированным расположением элементов

На мобильных устройствах:
- Холст автоматически масштабируется под размер экрана
- Чат и панель игроков располагаются вертикально
- Кнопки и инструменты увеличены для удобства нажатия
- Панель инструментов прокручивается горизонтально

## Как играть

1. Введите ваше имя на главной странице
2. Оставьте поле "ID комнаты" пустым для создания новой комнаты или введите существующий ID
3. Нажмите "Присоединиться"
4. Дождитесь минимум 2 игроков
5. Нажмите "Начать раунд"
6. Первый игрок выбирает слово из двух предложенных
7. Рисуйте выбранное слово, остальные угадывают через чат
8. Первый, кто наберет 100 очков, побеждает

## Инструменты рисования

- Кисть - основной инструмент рисования
- Ластик - стирание
- Заливка - заполнение области цветом
- Линия - рисование прямых линий
- Прямоугольник - рисование прямоугольников
- Круг - рисование кругов
- Размер - регулировка толщины линии (1-50)
- Палитра - выбор цвета из предустановленных или через color picker
- Очистить - полная очистка холста
- Отменить - отмена последнего действия

## Особенности

- Если вы напишете однокоренное слово, система покажет вам "Очень близко!", но другие игроки увидят только "***"
- Чем быстрее угадаете слово, тем больше очков получите
- Первый угадавший получает больше очков, чем последующие
- Рисующий получает 10 очков за каждого угадавшего игрока
- Время на выбор слова: 15 секунд (если не выбрали, система выберет случайное)
- Время на рисование: 80 секунд

## Остановка сервера

Нажмите Ctrl+C в терминале для остановки сервера.
